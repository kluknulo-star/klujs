# Начало работы с js и Vue.js
Собственно, проект демонстрирует простенькое SPA приложение, в котором реализован:
- CRUD пользователей (на бэке)
- vue-router (позволяет без перезагрузки менять url в браузере)
 
## Запуск проекта

```cp .env.example .env```

```composer install```

```npm run dev```
